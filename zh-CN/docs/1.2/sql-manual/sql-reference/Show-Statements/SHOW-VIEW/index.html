SHOW-VIEW

SHOW-VIEW​

Name​

SHOW VIEW

Description​

该语句用于展示基于给定表建立的所有视图

语法：

 SHOW VIEW { FROM | IN } table [ FROM db ]

Example​

  1. 展示基于表 testTbl 建立的所有视图 view

    SHOW VIEW FROM testTbl;

Keywords​

SHOW, VIEW
